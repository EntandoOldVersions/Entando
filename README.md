## Also, what is needed to put together a development environment?

* Java JDK 1.6 
* Maven 2.2.1
* Ant 1.8+
* Ant Contrib
* PostgreSQL 8.3+ or MySQL 5.0+

### Optional

* Eclipse Indigo with m2eclipse, WTP, and m2e-wtp (m2e-wtp 0.15+)
* Netbeans 7.0.1+
